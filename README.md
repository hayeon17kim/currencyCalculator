# 환율계산기 - 프론트엔드
이 프로젝트는 최초에 [Create React App](https://github.com/facebook/create-react-app) 기반으로 만들었습니다.

D조의 REACT 학습용 토이 프로젝트 입니다.

## 사용 가능한 스크립트
1.설치
### `yarn install`
2.실행
### `yarn start`
3.빌드
### `yarn build`
4.린트체크
### `yarn lint`
5.전체 자동픽스
### `yarn lint-fix`

node 기반 백엔드 API 프로젝트는 [API](https://github.com/xlael/currencyCalculator-api) 를 참고 해 주세요.
